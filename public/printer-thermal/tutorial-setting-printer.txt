2. DOWNLOAD RECTA-HOST (UNTUK PRINTER KE PC/LAPTOP) & ZAGDIG (KHUSUS WINDOWS UNTUK INSTALL DRIVER KE USB PRINTERNYA)
3. UNTUK PRINT DENGAN LAPTOP/PC MENGGUNAKAN USB

JIKA MENGGUNAKAN WINDOWS (IKUTI 4-8) :
4. BUKA APLIKASI ZAGDIG.EXE (SILAHKAN DOWNLOAD)
5. PILIH MENU OPTION, CHECK "LIST ALL DEVICE". MAKA AKAN MUNCUL SEMUA DEVICE PADA LAPTOP/PC
6. COLOKAN USB PRINTER KE LAPTOP/PC 
7. PILIH USB PRINTERNYA (BIASANYA NAMANYA THERMAL ... , JIKA RAGU SILAHKAN COLOKAN ULANG DAN LIHAT DEVICE YANG MUNCUL PADA DROPDOWN)
8. JIKA SUDAH KLIK "Install Driver" / "Replace Driver"

9. INSTALL RECTA-HOST (SILAHKAN DOWNLOAD YANG BERFORMAT .EXE)
10. JIKA MUNCUL PERINGATAN MORE INFO > KLIK RUN ANYWAY
11. TUNGGU SAMPAI LOADING SELSAI, JIKA BERHASIL AKAN TERBUKA OTOMATIS
12. PILIH "USB" (PRINTER ADAPTER)
12. ISIKAN APP KEY BEBAS / DEFAULT SAJA
13. UNTUK PORT "1811" JIKA INGIN MENGUBAH PASTIKAN TIDAK BENTROK DENGAN PORT LAINNYA
14. KLIK SAVE & START
15. HUBUNGKAN PRNTER "APP KEY" & "PORT" PADA APLIKASI . 
16. SELESAI
